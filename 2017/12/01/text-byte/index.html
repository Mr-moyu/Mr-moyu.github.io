字节概要

  • bytes 对象可以从str对象使用给定编码构建
  • 各个元素是range(256)内的整数
  • bytes 对象切片还是bytes 对象
  • bytes 是不可变对象，bytearray 是可变序列
  • bytearray 对象没有字面量句法，而是以bytearray() 和字节序列字面量参数形式显示，就像一个函数
  • bytearray 对象的切片还是bytearray 对象
